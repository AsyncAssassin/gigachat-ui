# GigaChat UI (Frontend Homework)

Учебный интерфейс чата на React + TypeScript + Vite.

## Текущий статус проекта

Проект адаптирован под домашнее задание по теме "Состояние компонентов и хуки":

- локальное состояние сообщений в `ChatWindow` через `useState`
- `isLoading` для состояния ожидания ответа ассистента
- симуляция ответа ассистента через `setTimeout` (1-2 секунды)
- `TypingIndicator` во время ожидания
- управляемый `InputArea` с очисткой после отправки
- блокировка ввода и кнопки отправки при `isLoading` и пустом тексте
- автоскролл к последнему сообщению (`useRef` + `useEffect`)
- сохранение истории отдельно для каждого чата по `chatId` при переключении

Дополнительно оставлены:

- боковая панель чатов (создать, выбрать, удалить, редактировать)
- markdown-рендеринг сообщений (`react-markdown` + `remark-gfm`)
- панель настроек
- адаптивная верстка (desktop/mobile)
- светлая/тёмная тема на CSS-переменных

## Стек и версии

- React `19.2.0`
- TypeScript `5.9.x`
- Vite `7.3.x`
- Vitest `4.1.x`
- ESLint `9.x`

## Проверка и тесты

В проекте есть unit-тесты (Vitest + Testing Library), включая:

- тесты стриминговых хуков
- тест на сценарий переключения чатов во время ожидания ответа

Запуск:

```bash
npm test
```

## Запуск проекта

Требования: Node.js 20+.

```bash
npm install
npm run dev
```

Приложение будет доступно на `http://localhost:5173`.

## Скрипты

```bash
npm run dev
npm run lint
npm run test
npm run test:watch
npm run build
npm run preview
```

## Ограничения текущего этапа

- нет реального backend/LLM API (используются моки и таймеры)
- история не сохраняется после перезагрузки страницы (runtime-only)
