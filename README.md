# GigaChat UI (Frontend Homework)

Учебный интерфейс чата на React + TypeScript + Vite.

## Что реализовано

- React + TypeScript приложение
- моковые данные для чатов и сообщений (без API)
- стилизация через CSS Modules
- светлая/тёмная тема на CSS-переменных
- адаптивная верстка (mobile + desktop)
- markdown-рендеринг сообщений через `react-markdown`
- базовые действия с чатами: создать, выбрать, удалить, редактировать
- панель настроек с сохранением состояния в runtime

## Ограничения текущего этапа

- без реального backend/API
- без localStorage/IndexedDB
- без Redux/Zustand
- без тестов

## Запуск проекта

Требования: Node.js 20+.

```bash
npm install
npm run dev
```

Приложение будет доступно на `http://localhost:5173`.

## Полезные команды

```bash
npm run lint
npm run build
npm run preview
```
