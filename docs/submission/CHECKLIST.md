# Финальный чек-лист перед отправкой

## 1. Техническая готовность

- [ ] `npm run lint` прошел без ошибок.
- [ ] `npm run test` прошел без падений.
- [ ] `npm run test:server` прошел без падений.
- [ ] `npm run build` успешно собирает фронтенд.
- [ ] `npm run security:secrets` прошел без утечек.

- [ ] В репозитории нет реальных ключей и токенов.
- [ ] Все секреты хранятся только в `server/.env`.

- [ ] GitHub required check `gitleaks` зеленый в PR.

- [ ] Закрыты все обязательные критерии интерфейса чата.

- [ ] Streaming/SSE и fallback на REST работают стабильно.
- [ ] Демо-сценарий multimodal (изображение) работает стабильно.

## 4. Материалы для преподавателя

- [ ] README полностью заполнен на русском.
